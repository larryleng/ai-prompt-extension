# 大西瓜浏览器插件

一个功能强大的浏览器AI提示词扩展插件，帮助用户优化和扩展提示词内容，提升AI对话体验。

## ✨ 功能特性

- 🤖 **AI提示词扩展** - 智能优化和扩展用户输入的提示词
- 🌐 **多语言翻译** - 支持中英互译
- 📰 **新闻资讯浏览** - 获取最新的AI和技术资讯、实用工具
- 💾 **提示词保存管理** - 保存和管理常用的提示词模板
- 🎨 **自定义模板** - 支持图片和视频提示词模板定制
- 🌙 **深色/浅色主题** - 支持主题切换，保护视力
- ⚙️ **多API支持** - 兼容OpenAI、Claude、本地Ollama等多种AI服务

## 🚀 安装方法

### 方法一：直接安装
1. 下载项目文件到本地
2. 打开Chrome浏览器，进入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹完成安装

### 方法二：从Release下载
1. 前往 [Releases页面](../../releases) 下载最新版本
2. 解压下载的文件
3. 按照方法一的步骤3-5进行安装

## 📱 使用说明

### 基础配置
1. 点击浏览器工具栏中的插件图标
2. 在弹出窗口中配置您的API密钥和端点
3. 选择合适的AI模型

### 提示词扩展
1. 在主界面输入您的提示词
2. 提示词管理
3. 自定义提示词工程

### 翻译功能
1.中英文互译

## 🛠️ 技术栈

- **前端**: JavaScript (ES6+), HTML5, CSS3
- **API**: Chrome Extension API
- **数据库**: Supabase (用于业内行情和模板数据)
- **AI服务**: 支持OpenAI、Claude、Ollama等
