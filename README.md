# AI Prompt Extension

一个轻量的浏览器扩展，用于管理提示词、快速生成内容，并集成多种数据源展示（新闻、模板等）。适合创作者、产品经理和开发者日常使用。

## 特性
- 提示词管理：保存、搜索、分类和快速插入
- 扩写/改写：支持多模型（需正确配置API）
- 模板管理：按视频/图片/文案等场景组织模板
- 新闻视图：从配置数据源展示每日更新
- 统一提示：所有提示在输入框中心显示，3秒后自动淡出
- 错误增强：API配置错误等会提供清晰原因和解决建议

## 安装
1. 克隆仓库或下载源码
2. 打开浏览器扩展开发者模式（Chrome: `chrome://extensions/`）
3. 选择“加载已解压的扩展程序”，指向项目根目录

## 使用
- 点击弹窗页进入配置，设置 API Key 与模型等参数
- 在主界面中输入提示词，使用“扩写”等功能
- 在“已保存提示词”页管理常用提示及分类

## 开发
- 本地预览：`cd pages && python -m http.server 8000`，访问 `http://localhost:8000/`
- Node 已安装，可用于后续脚本或打包

## 目录结构
- `manifest.json` 扩展声明
- `pages/` 前端页面与脚本
- `pages/modules/` 业务模块（utils、api、data、ui等）
- `config/` 缓存与配置文件
- `images/` 图标与图片资源
